# Receipt Examples & Format Guide for Itay

## Receipt Format

Each receipt in your expense tracker should follow this simple YAML frontmatter structure:

```
---
date: YYYY-MM-DD
vendor: [Business/Store Name]
amount: [X.XX ILS]
category: [Category Name]
notes: [Optional short note]
---
```

**Important Note**: The amounts below are examples and estimates based on typical Tel Aviv prices (April 2026). When you add your own receipts, use the actual amounts from your receipts. This guide is just to show you the format and realistic spending patterns for your lifestyle.

## Suggested Monthly Budget Breakdown (Tel Aviv, 2026)
*Rent excluded*

| Category | Budget | Notes |
|----------|--------|-------|
| **Groceries** | 800-1,000 ILS | Cooking at home 60% of the time |
| **Restaurants & Cafes** | 600-800 ILS | Lunch 3x/week at office, dinner out 2x/week |
| **Transport** | 250-300 ILS | Monthly bus pass + occasional taxi/Gett |
| **Subscriptions** | 800-1,000 ILS | Netflix, Spotify, iCloud, AWS (learning), GitHub Pro, etc. |
| **Utilities** | 600-700 ILS | Electricity + internet + mobile |
| **Health** | 300-400 ILS | Gym (320 ILS) + occasional pharmacy |
| **Entertainment** | 400-500 ILS | Bars, cinema, escape rooms, events |
| **Clothes** | 300-400 ILS | Occasional purchases (Zara, H&M) |
| **Tech & Accessories** | 200-300 ILS | Phone accessories, cables, peripherals |
| **Miscellaneous** | 200-300 ILS | Unexpected expenses |
| | | |
| **TOTAL (monthly)** | **4,450-6,400 ILS** | Depends on dining frequency & entertainment |

**Notes**:
- This assumes you cook most meals and eat out for lunch/dinner a few times weekly
- Subscriptions are high because you likely have work-related subscriptions (AWS, GitHub, Slack, etc.)
- Budget can be reduced by 500-800 ILS if you meal-prep and reduce dining out
- Utilities are seasonal (higher in summer for AC, winter for heating)
